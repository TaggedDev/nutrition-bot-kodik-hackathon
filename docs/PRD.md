# PRD: AI Food Logging Assistant

## Бизнес-проблема
Пользователи до сих пор тратят несколько минут на ручной поиск продуктов и калькуляцию КБЖУ по каждому блюду. Это делает журнал питания рутинным и ненадёжным, особенно когда надо быстро зафиксировать приём пищи. Приложение должно убрать этот барьер, позволяя записывать рацион за 10–20 секунд без ручного ввода каждого ингредиента.

## MVP
- Мультимодальный ввод: текст, голос, фото упаковки, ценника и штрихкода, а также фото блюда.
- Разбор фразы/голоса в структурированный JSON с продуктами, порциями и приёмом пищи.
- Поиск КБЖУ через внешние базы (Open Food Facts, USDA) и локальную историю.
- Автоматический расчёт КБЖУ с confidence score и вынос кратких уточняющих вопросов только при нехватке критичных данных.
- Сохранение записи с источником данных, контекстом, порцией и возможностью правки.
- Рекомендации по следующему приёму с учётом целей, предпочтений и прогресса дня.

## User stories
1. **Скоростной логгер.** Как занятый профессионал, я хочу записать еду голосом или текстом за несколько секунд, чтобы не терять ритм дня.
2. **Прозрачный расчёт.** Как заботящийся о данных пользователь, я хочу видеть источник КБЖУ, порцию и confidence score, чтобы доверять записи и быстро исправлять неточности.
3. **Помощник по меню.** Как пользователь, я хочу получать рекомендации, что съесть дальше на основе уже съеденного и целей, чтобы не выходить из рекомендованной калорийности.
4. **Кросс-платформенная доступность.** Как владелец разных устройств, я хочу использовать один PWA интерфейс на телефоне и компьютере.
5. **Сохранённые продукты.** Как постоянный пользователь, я хочу видеть историю и сохранённые блюда, чтобы ускорить повторное логирование.

## Что не входит в MVP
- Полностью автоматическое распознавание блюд по фотографии без подтверждения пользователя.
- Экспертные медицинские расчёты микронутриентов и биомаркеров.
- Интеграция с внешними wearables и медицинскими API (появится позже).
- Обширная система рекомендаций на уровне персональных диет/планов — пока только быстрые подсказки в контексте дня.
- Интеллектуальное автопозаполнение дневника (например, планирование на неделю вперед).
